解决VS Code保存时候自动格式化,并且光标会自动跳转到文档最后面

不知道从哪个版本起，VS code保存会自动格式化。以前都是alt+shift+F格式化的，现在一保存就格式化，react项目也没法写，如何解决呢，操作如下：
于是我找到设置，搜索关键字：格式化或者保存，然后配置了两个参数， “editor.formatOnType”: false,和”editor.formatOnSave”: false 重启，发现不管用，依然会保存之后自动格式化。

把这个插件给禁用掉就不会自动保存了。
