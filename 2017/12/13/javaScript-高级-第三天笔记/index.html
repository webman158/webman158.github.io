注意点

3 参数可省略
4 最后一个参数，表示函数体function(){}花括号里面的代码
5 如果新创建的函数没有参数，那么第一个就是函数体（如果只有一个参数，那么这一个就是函数体）

eval

  • 语法：eval(‘字符串’);
  • 作用：这个函数在调用的时候会将字符串参数作为js代码来执行

能够解释代码字符串的能力非常强大，但也非常危险。因此在使用 eval() 时必须极为谨慎，特别是在用它执行用户输入数据的情况下。否则，可能会有恶意用户输入威胁你的站点或应用程序安全的代码（即所谓的代码注入）。

function的相关属性和方法

  • name

    函数名.name：获取函数名称
    获取到的名字是 字符串类型的
    注意：IE浏览器不支持

  • length

    函数名.length
    作用：获取形参的个数

  • func.caller（废弃）：获得调用函数的函数引用（用来获取当前函数是在哪个函数中被调用的）

  • arguments 重点，常用

    作用：用来获取传入的实参列表
    类型：对象类型，是一个伪数组
    注意：只能在函数内部使用(类似 return 也是只能在函数内部使用一样)
    而且每一个函数中都有一个 arguments 对象
    可以把 arguments 看作是函数内部的局部变量

Object.prototype 常用成员

  • constructor

    指向了当前的构造函数 Object

  • hasOwnProperty

    • 作用：
      判断有没有指定的属性
      1 如果属性是对象自身提供的，返回true
      2 否则，返回false
    • 语法：
      布尔值 = 对象.hasOwnProperty(‘属性名’)
  • 该方法常用场景

  • isPrototypeOf

    • 作用：
      判断前面对象是不是参数对象的原型对象，如果是就返回：true，否则返回false
      只要前面的对象在参数对象的原型链上，结果就为true（注意，原型链不包括头（对象本身）尾（null）两个）
      与 instanceof 运算符的作用相同！
    • 语法：
      布尔值 = 原型对象.isPrototypeOf( 实例对象 )

  • propertyIsEnumerable

    • 作用:
      1 该方法表示属性必须是 自己提供的, 同时可以被枚举（遍历）的, 那么就返回 true
      2 否则返回 false
    • 语法：
      布尔值 = 对象.propertyIsEnumerable(‘属性’)
    • 注意：
      在 ES3 的基础上, 该方法与 hsOwnProperty 的作用相同
      在 ES5 中引入了 对象特性( attribute ) 的概念, 该方法的作用才体现出来。
