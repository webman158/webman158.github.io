函数四种调用模式

  • 1 函数模式
  • 2 方法模式
  • 3 构造器模式
  • 4 上下文（借用方法）模式

    以上4种模式，就是根据函数内部 this 的指向不同来划分的！

前三种调用模式的区别

  • 函数调用模式中的 this 指向：全局对象（window）
  • 方法调用模式中的 this 指向：调用方法的对象
  • 构造器模式 中的 this 指向：新创建出来的对象

上下文（借用方法）模式

  • 上下文，即环境，用于指定方法内部的 this
  • 上下文调用模式中，this可以被随意指定为任意对象

两种实现方式

  • 1 函数名.apply( … )
  • 2 函数名.call( … )
  • 作用：调用方法（为了改变函数内部this的指向）
  • call / apply 这两个方法是由： Function.prorotype 提供的
  • 注意：方法还是会被调用的，只不过会根据参数来改变this的指向以及传参
