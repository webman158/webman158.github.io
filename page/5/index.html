win7/10常用快捷键

Veröffentlicht am 2015-12-27

win7、10快键键

  • 单独按Windows：显示或隐藏 “开始”功能表
  • Windows+BREAK：显示“系统属性” 对话框
  • Windows+D：显示桌面
  • Windows+M：最小化所有窗口
  • Windows+Shift+M：还原最小化的窗口（在Windows+M操作之后才起作用）
  • Windows+E：开启“资源管理器”
  • Windows+F：查找文件或文件夹 (win10可能不支持)
  • Windows+R：开启“运行” 对话框
  • Windows+L：锁屏
  • windows + 上下左右箭头: 控制分屏，屏幕位置
  • windows + tab: 桌面所有应用程序（win10可以有多个桌面）
